# 装箱与拆箱

# 封装类

所有的基本类型，都有对应的类类型
比如 int 对应的类是 Integer
这种类就叫做封装类

# 10.2 Number 类

数字封装类有

  • Byte Short Integer Long
  • Float Double

这些类都是抽象类 Number 的子类

# 基本类型 <-> 封装类型

  • 基本类型转换成封装类型
  • 封装类型转换成基本类型

# 自动装箱与拆箱

  • 自动装箱：通过 = 符号自动把 基本类型 —> 类类型

  • 自动拆箱：与自动装箱相反，通过 = 符号自动把 类类型 —> 基本类型

# int 的最大值 / 最小值

int 的最大值可以通过其对应的封装类 Integer.MAX_VALUE 获取；

同理，最小值通过 Integer.MIN_VALUE 获取；
