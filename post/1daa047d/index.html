# GitCode 图床

GitCode 实际上是使用 Gitlab 服务搭建的一个代码托管平台，因此我们可以使用【Gitlab】图床插件来将图片上传到 Gitcode。而从 npm 官网上正好可以找到这样的插件：

注意：推荐使用第一个插件 picgo-plugin-gitlab-files ，这个插件上传的图片存放在自己的仓库里，另一个好像不是存放在自己的仓库。

另外，听说又拍云作为图床也很不错，不过由于又要注册又要实名认证的，以后再弄吧。（Gitcode 与 CSDN 账号关联，不用重新注册）
